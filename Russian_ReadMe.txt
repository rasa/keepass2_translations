KeePass 2.32:

! Изменен перевод обозначения поля для имени пользователя
  с "Имя" на "Логин". Это сделано для простоты разделения
  понятий: ранее имя использовалось в нескольких значениях
  при переводе. Если возникают проблемы с таким переводом,
  пишите мне на адрес keepass.ru@gmail.com (Андрей).
! Перевод пункта меню "Открыть ссылку" изменен на "Открыть
  по ссылке".

- Поле быстрого поиска (на панели управления главного окна)
  позволяет искать использую регулярные выражения. Для этого
  необходимо заключить регулярное выражение в "//". Например,
  при поиске по "//Саша|Женя//" будут показаны все записи,
  содержащие "Саша" или "Женя".
- Добавлена закладка "Дополнительно" в диалог "Открыть по
  ссылке" (содержимое закладки может меняться модулями).
  Добавлены такие поля: тайм-аут, преаутентификация, агент
  пользователя (user-agent), поведение по ответу кода 100,
  пассивный режим FTP.
- Добавлено определение режима браузера: меню пуска перед
  логином пользователя.
- При выборе записи в главном списке записей, ее группа
  автоматически выделяется в дереве групп.
- Соответствие для автонабора: добавлена настройка "Просроченные
  записи могут соответствовать". Она выключена по умолчанию,
  т.е. просроченные записи будут игнорироваться.
- Добавлена настройка "Всегда показывать окно выбора записи
  для глобального автонабора". Она выключена по умолчанию. Если
  ее включить, то окно выбора записи будет появляться всегда,
  даже если не найдено ни одной записи, или найдена одна запись.
- Добавлен шаблон {GROUP_NOTES} для автонабора.
- Добавлена возможность импорта NPW файлов nPassword 1.0.2.41.
- Модуль импорта/экспорта в триггерах и KPScript теперь можно
  указывать как по имени, так и по расширению.
- При запуске под .NET 4.5 и выше, безопасные соединения
  (например, для WebDAV) теперь поддерживают TLS 1.1 и TLS 1.2
  в дополнение к SSL 3 и TLS 1.0.
- Добавлено исправление для Mono: при запуске на
  Unity или Pantheon, KeePass не будет пытаться добавить иконку
  на системной панели. Если она нужна, необходимо установить
  модуль, содержащий application indicator.
- Исправление для Mono: нет поддержки свойства
  SystemInformation.SmallIconSize для Mac OS X.
- Добавлен параметр "-wa-disable:" для коммандной строки для
  отмены действия исправления для Mono. Можно перечислить
  список идентификаторов для отмены, разделенных запятой.
- KPScript: если значение параметра "-ref-*:" заключено в "//",
  то оно будет считаться регулярным выражением, которое будет
  использоваться для поиска соответствий в полях записи.
- KPScript: .NET 4.0/4.5 является предпочтительной если установлена.
- KPScript: улучшена поддержка устройств с высоким разрешением.
- Автонабор: улучшена совместимость с целевыми окнами, которые
  медленно обрабатывают активацию и игнорируют любой ввод до
  готовности, например, Microsoft Edge.
- При редактировании строковых записей фокус устанавливается
  на поле ввода значения.
- Улучшена фильтрация переводов строк в панели просмотра записи.
- Улучшено сообщение об ошибке при использовании учетной записи
  Windows в составе составного основного пароля.
- "О программе" теперь содержит информацию об использовании
  версии для разработчиков.
- Изменен сертификат, которым подписан код.
- Обновлен инсталятор.
- Различные оптимизации кода.
- Другие мелкие улучшения.
- После некорректного завершения операции перетягивания,
  восстанавливается предыдущая выбраная группа.

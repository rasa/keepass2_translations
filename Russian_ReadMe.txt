KeePass 2.34:

- Файл с информацией о версии (который скачивается для проверки
  наличия новой версии) теперь подписан используя RSA / SHA-512,
  более того, он скачивается через HTTPS.
- Добавлена настройка "Блокировать при сворачивании главного окна
  в системную панель".
- Добавлена настройка "Esc сворачивает в системную панель вместо
  блокировки".
- Добавлена комбинация клавиш Ctrl+Q для закрытия KeePass, как
  альтернатива для Alt+F4.
- Добавлен бит отключения пункта меню "Проверка наличия
  обновлений" в UIFlags.
- Модули: добавлена поддержка подписанных файлов с информацией
  о версии.
- При закрытии базы паролей, теперь KeePass находит и удаляет
  все временные файлы, которые могли быть созданы и не удалены
  при неудачной попытке печати.
- Локальный файл справки: улучшена поддержка высокого разрешения.
- Различные оптимизации кода.
- Другие мелкие улучшения.
